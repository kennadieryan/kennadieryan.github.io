Kennadie Neal GIS Portfolio Website

How to use:
1. Unzip this folder.
2. Open index.html in a browser to preview.
3. To publish on GitHub Pages, upload all files in this folder to a GitHub repository.
4. In GitHub, go to Settings > Pages and publish from the main branch.
5. Submit the GitHub Pages URL, or upload this ZIP if your instructor accepts file upload.

Included:
- index.html
- style.css
- script.js
- project map preview images
- PDF project files
- resume and cover letter PDFs
